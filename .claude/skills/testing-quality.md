# Skill — Testing and Quality

Use esta skill ao implementar funcionalidades.

## Diretrizes

- Criar testes unitários para use cases.
- Criar testes e2e para fluxo principal.
- Testar cenários de erro.
- Rodar lint, tests e build antes de finalizar.

## Cenários obrigatórios

- Criar veículo com sucesso.
- Rejeitar veículo com preço inválido.
- Listar disponíveis ordenado por preço.
- Comprar veículo autenticado.
- Bloquear compra sem token.
- Bloquear compra duplicada.
- Listar vendidos ordenado por preço.
